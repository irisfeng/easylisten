# 轻听路线图:迭代 · 用户培养 · 付费

原则:**里程碑用"信号"触发,不用日期**。每一步只在上一步的证据出现后
启动;数据说话,宁慢毋滥——产品哲学同样适用于路线图本身。

当前基线(2026-07):每日 2–6 篇 + 周六深读 + 主编的话;MiniMax 2.8
超拟人双声;整篇零停顿;最多两篇高门槛双语实验;分享卡;45 源(中文约 33%);
完听率/分享埋点已铺;月成本 ≈ 70 元。

---

## 一、迭代优化线(产品与技术)

### 近期(数据驱动的收尾)

| 事项 | 触发条件 / 依据 |
|---|---|
| 品味调优:朋友反馈 → rubric 逐条修订 | 持续,每次小步 |
| 新源验活与淘汰(分层权重、中文覆盖、馈源可用性) | 每日 ingest 日志 + `test:curation` |
| 两周完听率复盘:定篇数上限、深读节奏、EN 去留 | ~08-01,Vercel Analytics |
| 偏好云同步激活(代码已休眠合入) | 用户建 Supabase(3 分钟,见 prefs-sync.md) |
| iOS TestFlight 首个构建 | 用户 Mac 空闲时,docs/ios-app.md |

### 中期(体验纵深)

- **存档与检索**:内容过百篇后,按期归档页 + 话题聚合页(听过的自动置灰)
- **合集**:编辑部把往期串成主题合集("宇宙五讲"),长青内容二次分发
- **RSSHub 自建**:豆瓣公共实例失活或想上晚点 LatePost 时(半天)
- **音频架构**:仓库音频超 ~1GB 时迁对象存储(R2/OSS),git 只存清单
- **英文轨去留**:listen_en 两周数据裁决——扩成正式分区或摘除

### 远期(平台)

- App 正式上架(美区先行,国内需 ICP 备案)
- 个性化召回(账号体系之后):每人一份"为你补课"的旧文队列

## 二、用户培养线(增长与留存)

增长模型:**分享卡获客 → 首听转化 → 连续 7 天留存 → 习惯养成**。
小众产品不投流、不裂变,靠内容与仪式感自然生长。

1. **获客**:分享卡是唯一主动增长机制——每篇好文章都是物料;
   朋友圈文案库(已备三版)按场景复用
2. **首听转化**(打开 → 听完第一篇):首页"今日刊"眉标 + 主编的话
   即导购;观察指标 = 首访完听率
3. **留存钩子**(已有,持续打磨):
   - 每天 6 点准时出刊 = 时间锚点
   - 周六深读 = 每周仪式
   - 主编的话 = 人格连接
   - 音色偏好/兴趣记忆 = 切换成本
4. **反馈回路**:朋友的每条吐槽进 rubric;分享数是品味投票
5. **克制清单**(刻意不做):推送轰炸、签到打卡、播放量展示、
   算法无限流——都与"安静地听点好东西"相悖

**健康指标**(Vercel Analytics,两周看一次):
周回访设备数、人均完听篇数、分享卡发送数。北极星 = **每周听完
≥3 篇的设备数**(核心习惯用户)。

## 三、付费订阅线

**大门槛(未达不启动)**:核心习惯用户(周听完 ≥3 篇)稳定 ≥ 50 人,
且自然增长持续 4 周。在被需要之前伸手收钱,是小众产品最常见的死法。

### 分层(免费养口碑,付费卖纵深与便利)

| 权益 | 免费 | 会员(预估 ¥15/月 或 ¥128/年) |
|---|---|---|
| 每日精选 + 主编的话 | ✓ 全部 | ✓ |
| 存档 | 最近 7 天 | 全部 + 合集 |
| 周末深读 | 试听前 1/3 | 全文全音频 |
| 音色 | 默认女声 | 双声切换 + 后续新音色 |
| 私人播客 feed | — | 专属 RSS,播客 App 直接订阅(杀手锏) |
| 双语精听 | 每日最多 2 篇高分稿 | 全部(若英文轨验证成立) |

### 实施顺序(启动后 2–3 周)

1. **收款**:爱发电/面包多卖兑换码(个人可收款,零资质)→ 有主体后
   接微信支付;海外 Stripe
2. **架构**:内容仓库转私有;会员内容经带鉴权 API/签名 URL 下发;
   部署迁 Cloudflare Pages(Vercel Hobby 禁商用)或升 Vercel Pro
3. **账号**:兑换码激活 = 轻账号(邮箱或码即身份),在匿名设备 ID
   之上生长,不推倒重来
4. **播客 feed**:每会员一个带 token 的 RSS 地址,复用现有音频管线

### 成本与回本参考

现成本 ≈ 70 元/月;付费启动后新增(Vercel Pro 或 CF + Supabase +
带宽)≈ 100–200 元/月。**约 15–20 个年费会员即覆盖全部成本**,
之后每一分都是内容预算(更好的音色、更多的源)。
